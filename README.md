QC&I Central Laboratory Website - Hosting Ready Package

Files included:
1. index.html  -> Main website file
2. README_HOSTING.txt -> Basic hosting instructions

How to publish quickly with Netlify:
1. Go to Netlify
2. Drag and drop index.html
3. Netlify will instantly publish the site
4. Copy the public URL
5. Share that URL back to generate the final live QR code

How to publish with GitHub Pages:
1. Create a new GitHub repository
2. Upload index.html to the repository root
3. Open repository Settings > Pages
4. Enable GitHub Pages from the main branch
5. Copy the published public URL
6. Share that URL back to generate the final live QR code

Notes:
- This package is self-contained
- Images and QR are embedded inside index.html
- No separate asset folder is required
